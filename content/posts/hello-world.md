Welcome to the Kaonashi theme demo! This post demonstrates the terminal-inspired aesthetic with monospace typography and a dark/light mode toggle.

## Features

- **Dark/light mode** — Toggle with the sun/moon button in the header
- **Monospace typography** — Clean, readable code-friendly fonts
- **Responsive design** — Works on mobile, tablet, and desktop

### Code block example

```rust
fn main() {
    println!("Hello, Kaonashi!");
    for i in 0..5 {
        println!("Count: {}", i);
    }
}
```
